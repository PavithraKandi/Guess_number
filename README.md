# 🎯 Guess Number Game

A simple and interactive number guessing game built using **HTML**, **CSS**, and **JavaScript**.

## 📌 Overview

The Guess Number Game generates a random number between **1 and 100**. The player must guess the correct number within a limited number of attempts. After each guess, the game provides hints indicating whether the guessed number is too high or too low.

## 🚀 Features

* Random number generation using JavaScript
* User-friendly interface
* Real-time feedback for each guess
* Attempt counter
* Success and failure messages
* Game over after maximum attempts
* Responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎮 How to Play

1. Enter a number between **1 and 100**.
2. Click the **Check** button.
3. The game will tell you whether the target number is higher or lower.
4. Continue guessing until you find the correct number or run out of attempts.
5. Try to guess the number in the fewest attempts possible.
